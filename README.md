## 📚 기술스택
<br>

| 분야           | 사용 기술                       |

| BackEnd        | Django |
| Database       | SQLite              |
<br>

## 📁 주요 기능

- 회원가입, 이메일 로그인, Kakao 로그인, Google 로그인
- 로그인 사용자와 댓글 작성자의 아바타
- 홈페이지(최근 등록된 3개의 상품)
- 상품 등록, 수정, 삭제 (등록, 수정 -> 마크다운 작성)
- 카테고리 
- 태그
- 검색
- 댓글 등록, 수정, 삭제(본인만 가능)
